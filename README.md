# Gerador de Conselhos

Um pequeno aplicativo web que exibe conselhos aleatórios usando a API [Advice Slip](https://api.adviceslip.com/).

## Estrutura

- `index.html` – interface principal com o card de conselho e botão
- `styles.css` – estilos básicos responsivos
- `script.js` – lógica para buscar um conselho via `fetch` e atualizar a página
- `assets/` – imagens usadas no layout

## Como usar

1. Abra `index.html` em um navegador moderno.
2. Clique no botão com o dado para carregar um novo conselho.
3. O texto e o número do conselho são atualizados automaticamente.

## Tecnologias

- HTML
- CSS (flexbox, responsivo)
- JavaScript (fetch API, async/await)
